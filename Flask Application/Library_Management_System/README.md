# 📚 Library Management System

A full-stack Library Management System built using Flask, MySQL, JWT Authentication, HTML, CSS, Bootstrap, and REST APIs.

---

# 🚀 Features

✅ Admin Login with JWT Authentication  
✅ Books Management  
✅ Clients Management  
✅ Issued Books Tracking  
✅ Late Fee Calculation  
✅ Accounts Dashboard  
✅ REST APIs  
✅ MySQL Database Integration  
✅ Responsive Dashboard UI  

---

# 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Flask | Backend Framework |
| MySQL | Database |
| SQLAlchemy | ORM |
| JWT | Authentication |
| HTML/CSS | Frontend |
| Bootstrap | UI Design |
| JavaScript | Frontend Logic |
